This is a programme for inputting tweet archive data into excel file.\
This programme works on MacOS.

Follow the instructions bellow.
1. Download your Twitter account data from Twitter.
2. Find a file named \'91tweet.js\'92 from the downloaded file.
3. Change its file extension from '.js' to '.txt'.
4. Move the text file and this programme file into a same folder.
5. Run this programme on terminal. Do not forget to move to the folder that the text file and this programme stored before running the programme. Input the filenames with its file extension.

Reference
How to download your tweet data: https://help.twitter.com/en/managing-your-account/accessing-your-twitter-data
